Compliance & security posture
- We follow NIST and FedRAMP guidance as applicable and map controls to NIST SP 800-171 / 800-53 where required.

- Responsible AI: model governance, bias/fairness audits, explainability artifacts (model cards), and independent reviews integrated into development lifecycle.
